# Customer Churn Prediction in E-Commerce using Machine Learning

## 📌 Project Overview

This project aims to predict customer churn in an e-commerce platform using Machine Learning techniques. The objective is to identify customers who are likely to stop purchasing products so that businesses can take proactive retention measures.

The project applies LRFM (Length, Recency, Frequency, Monetary) analysis, K-Means clustering, feature engineering, and multiple Machine Learning algorithms to classify customers into churn categories and compare model performance.

---

## 🎯 Objectives

- Predict customer churn in an e-commerce platform.
- Perform customer segmentation using LRFM analysis.
- Apply feature engineering to improve model performance.
- Compare multiple Machine Learning algorithms.
- Evaluate models using standard performance metrics.
- Help businesses improve customer retention strategies.
